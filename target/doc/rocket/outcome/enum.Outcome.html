Enum rocket::outcome::Outcome

source · 
pub enum Outcome<S, E, F> {
    Success(S),
    Failure(E),
    Forward(F),
}
Expand description

An enum representing success (Success), failure (Failure), or forwarding (Forward).

See the top level documentation for detailed information.
